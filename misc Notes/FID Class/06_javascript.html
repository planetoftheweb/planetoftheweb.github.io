How do you use it?

Using the <script> tag

  • Better than inline, but not as maintainable
  • Code executes when script tags are read
  • 
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>JavaScript</title>
</head>
<body>
  <button id="hitme">Hello</button>
  <script>
    document.queryselector('#hitme').onclick=function {
        alert "I'm hit";
    }
  </script>
</body>
</html>

JavaScript data types

Understanding Variables

  • Can hold basic information 
    
var myName = 'Ray';
var myNumber = 3;
var myBoolean = true;
  • Can also be lists (arrays) 
    
var myList = ['Ray', 'Fred', 'Bonnie'];
var otherList = ['Ray', 3, false];
  • Have an index 
    
myList[1]; //Fred;
otherList[0]; //Ray;

JavaScript Objects
