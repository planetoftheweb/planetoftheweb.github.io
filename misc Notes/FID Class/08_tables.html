Using Tables

by Ray Villalobos

Use arrow keys to advance or esc

What are they for?

  • Tabular data
  • Not for layout

Basic Tables

  • Compound tag
  • <table> tag holds others
  • <tr> table rows
  • <th> header cell
  • Try it Add another table row

The colspan & rowspan attribute

  • Lets you make a cell span more than one td
  • Like merge cells in Excel
  • Total number of cells must still be the same
  • Colspan horizontal, rowspan vertical
  • Try it Experiment using the colspan attribute

The <caption> tag

  • Optional tag, describes the table
  • Must go immediately after the <table> tag
  • Only one caption per table
  • Try it Modify the caption

The <thead> tag

  • Lets you group header content
  • Use with <tbody> and <tfoot>
  • Inside the <table>, after the <caption> and before <tbody> or <tr>
  • Try it Use CSS to change the background color of thead

The <tbody> tag

  • Lets you group body content
  • Use with <thead> and <tfoot>
  • Inside the <table>, after the <caption> and <thead> elements
  • Try it Use CSS to change the style of tbody

The <tfoot> tag

  • Lets you group body content
  • Use with <thead> and <tbody>
  • Inside the <table>, after the <caption>, <thead> and before <tbody> or <tr>
  • Try it Experiment using the colspan attribute

The <th> tag

  • Identifies cells as header cells
  • Need the scope attribute as col or row
  • Display as bold and centered
  • Try it Experiment using the colspan attribute

Bootstrap Classes

  • Bootstrap adds a bunch of classes
  • table, table-striped, table-bordered, table-hover, table-condensed
  • Additional Contextual classes: active, success, info, warning, danger
  • Try it Experiment with Bootstrap table classes

Bootstrap Responsive

  • Wrap <div class="table-responsive"></div>
  • Table scrolls horizontally on small devices
  • Don't forget the viewport tag.
  • Try it Resize window to make it reponsive
